# mezo-hack

Hackathon demo apps for building on Mezo with x402.

## What's here

This repo hosts reference implementations that hackathon developers can clone,
run, and modify. Applications are drop-in demonstrations of the x402 payment
flow on Mezo Testnet.

> ℹ️ Content migration in progress. Demo apps arrive via [vativ/x402-mezo-preview#1](https://github.com/vativ/x402-mezo-preview/pull/1) relocation (tracked in repo-reorg epic `hq-ubcjh`).
